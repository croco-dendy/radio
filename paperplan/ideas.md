Mojo UI is the visual soul of my ecosystem, an independent NPM-package and design system built with React and Tailwind that recreates the physical feel of 1970s analog gear. This library features high-fidelity, animated components like metal toggle switches, brushed-aluminum cards, and functional VU-meters, all enhanced with satisfying auditory feedback (clicks and relays). It serves as the primary interface for all my apps, providing a tactile, "retro-future" aesthetic that stands out against modern, sterile design trends.

Instead of using external, cluttered platforms, Paper Plan lives directly in my project repository as a Bun-powered CLI and NPM package, using a structured universal source of truth.
